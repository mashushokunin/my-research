# Data

研究・実験で使用するデータセットを置くディレクトリです。

## Policy

- 大きな画像、動画、ROS bag、生成済みデータはGit管理しません。
- 小さな動作確認用データのみ `data/sample/` に置き、Git管理できます。
- 元データ、前処理中データ、処理済みデータは用途別に分けます。

## Suggested Layout

```text
data/
├── raw/        # 取得したままの元データ
├── interim/    # 前処理途中のデータ
├── processed/  # 実験に直接使う処理済みデータ
└── sample/     # 小さなサンプルデータ
```
